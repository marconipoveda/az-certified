- [x] Configure auto-shutdown for non-production VMs
- [x] Use Azure Advisor cost recommendations
- [ ] Purchase dedicated hosts
- [ ] Disable tagging

**Explanation:**
Auto-shutdown and Advisor insights manage fluctuating costs; dedicated hosts increase expense, and removing tags hinders allocation tracking.
