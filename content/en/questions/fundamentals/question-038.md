- [ ] Azure Firewall
- [x] Azure Key Vault
- [ ] Azure Information Protection
- [ ] Azure DDoS Protection

**Explanation:**
Key Vault stores keys, secrets, and certificates with rotation; firewall and DDoS provide network protection, not secret management.
