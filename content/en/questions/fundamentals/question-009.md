title: "Question 009"
question: "A financial institution deploys Azure Stack Hub inside its own datacenter so workloads run on single-tenant hardware under full regulatory control while consuming Azure-consistent services. Which cloud deployment model best describes this setup?"
---

> <a href="https://azure.microsoft.com/en-gb/resources/cloud-computing-dictionary/what-are-private-public-hybrid-clouds" target="_blank" rel="noopener noreferrer">https://azure.microsoft.com/en-gb/resources/cloud-computing-dictionary/what-are-private-public-hybrid-clouds</a>

1. [x] Private cloud using dedicated, single-tenant infrastructure
1. [ ] Public cloud because Azure Stack exposes Azure APIs
1. [ ] Hybrid cloud because the customer owns hardware on-premises
1. [ ] Community cloud because resources are shared among regulated entities
