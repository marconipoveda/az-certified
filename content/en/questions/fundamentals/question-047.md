- [ ] Azure RBAC
- [x] Privileged Identity Management (PIM)
- [ ] Azure Policy
- [ ] Azure Security Benchmark

**Explanation:**
PIM offers just-in-time activation and approvals for privileged roles; RBAC defines roles but not activation workflows.
